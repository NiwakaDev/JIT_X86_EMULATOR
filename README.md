# NIWAKA_JIT_X86
これはにわかなjit化されたx86エミュレータです。

※まだ、3つの機械語しかサポートしていません。
ツイートで広告したプログラムは動作しません。開始番地が本で変更されたからです。

modrm-test.binというプログラムが動きました。

```
BITS 32
    org 0x7c00
    sub esp, 16
    mov ebp, esp
    mov eax, 2
    mov dword [ebp+4], 5
    add dword [ebp+4], eax
    mov esi, [ebp+4]
    inc dword [ebp+4]
    mov edi, [ebp+4]
    jmp 0

```
